# LokasiKu
is an android application built with:

- **Kotlin**
- **MVVM Architecture**
- **Jetpack Compose**
- **Firebase Authentication**
- **Room**
- **Retrofit**
- **Shared Preference**
- **Node.js**
- **MySQL**
- **Google Maps API**
- **Android Studio**
- **Visual Studio Code**
